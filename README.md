This is a very simple excercise to practice CRUD architecture. 

The goal was to build a very simple Blog Interface where anyone could write and publish a simple text post. 

Tools used: 

* Ruby on Rails
* HTML
* CSS
* SCSS
* SQLite
* Javascript
